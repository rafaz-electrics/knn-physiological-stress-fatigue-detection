# knn-physiological-stress-fatigue-detection
Official repo: 'Implementasi Algoritma K-NN untuk Deteksi Tingkat Stres dan Kelelahan Berbasis Sinyal Fisiologis Tubuh'. K-NN pipeline classifying Normal, Stress, Fatigue using Heart Rate, GSR, Body Temp. Compares Z-Score &amp; Min-Max normalization validated via 10-Fold CV for model stability.
